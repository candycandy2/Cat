Android user download instructions

Support Android 5.0 and higher, On devices running Android 8.0 (API level 26) and higher, users must grant permission to install apps from a source that isn't a first-party app store.

1.Click Settings

2.Click to install

3.Allow for unknow APP install

In order for users to allow the installation of apps from non-first-party sources on devices running Android 7.1.1 (API level 25) and lower, they should enable the Unknown sources setting in Settings > Security

1. Click "Settings" to enter the settings page. Go to the settings page and then slide down to choose "Security".

2. Slide down to choose the "unknown source"

3. If popup of the message, press OK
