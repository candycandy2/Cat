iOS user download instructions

iOS Support version of 9.1 or more, please iOS users must read the relevant settings before downloading instructions:

1.If you are not logged out, press Cancel


2.Enter the settings, select General


3.Select Device Management


4.Select BenQ Corporation


5.Select Trust


6.Choose Trust for BenQ Corporation


7.When this screen appears, you can go back to the desktop to open QPlay


8.Open QPlay, allow push notification
